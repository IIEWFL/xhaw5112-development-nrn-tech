Empowering the Nation – Web and Mobile App
Overview
Empowering the Nation is a comprehensive digital platform developed to advance a local skills development initiative led by Precious Radebe. This platform is specifically aimed at supporting domestic workers and gardeners by providing them with accessible opportunities for skills training. Through both its website and mobile application, users can request information and obtain service quotes conveniently online.
The design of the platform ensures consistent user experience across both web and mobile interfaces. This consistency makes it easier for users to interact with the system, regardless of their preferred device. By leveraging technology, Empowering the Nation enhances accessibility for its users, fosters improved communication, and contributes to the growth and development of the community.

Key Features
The Empowering the Nation platform includes several core features designed to provide a seamless and supportive experience for users:
•	Home Page: The home page serves as the introduction to Empowering the Nation. It offers an overview of the initiative and provides direct links to summaries of available courses.
•	Course Summaries: Users can explore a selection of training programs, including both six-month and six-week options. Each summary includes a brief description of the program as well as the associated fees.
•	Course Details: Dedicated pages are available for each course. These pages give users access to comprehensive information about the training offered and include detailed pricing.
•	Fee Calculator: The platform features a fee calculator that enables customers to select multiple courses. The calculator automatically applies relevant discounts and provides a total cost calculation, including VAT.
•	Contact Page: Users can easily find contact information, such as addresses, phone numbers, emails, and the map locations of training venues.
•	Navigation: A drop-down menu and built-in error handling contribute to a smooth, user-friendly experience throughout the website and mobile app.
Design Motivation
The design of Empowering the Nation is guided by a user-centered philosophy that emphasizes simplicity, accessibility, and trust. The platform employs warm, professional colors and clear typography to reflect the values of empowerment, education, and community upliftment. Consistent layout and design elements across both the website and mobile app ensure that all users, regardless of their technical ability, can easily navigate and utilize the platform.
Logo
The logo for Empowering the Nation is crafted to symbolize growth, unity, and empowerment. It makes use of simple, modern shapes and uplifting colors to convey messages of hope and opportunity while maintaining a visually clean and non-overwhelming presence.
Technology Stack
The development of the platform relies on a robust set of technologies:
•	Frontend: HTML, CSS, and JavaScript are used to create an interactive and responsive user interface.
•	Backend: Depending on implementation needs, the backend may utilize Node.js, Firebase, or PHP.
•	Mobile App: The mobile application is developed using Kotlin within Android Studio.
•	Design Tools: Figma and Canva are employed designing wireframes and the logo.
 
How to Run the Project
For the Website
1.	Open the index.html file in your browser.
2.	Ensure all related CSS and JavaScript files are stored in the correct folders.
3.	Use the navigation menu to explore all sections of the platform.
For the Mobile App
1.	Open the project in Android Studio.
2.	Run the emulator or connect an Android device to your computer.
3.	Launch the app and test its features, including navigation, fee calculation, and contact forms.
